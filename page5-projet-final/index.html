Présentation du projet

Vous allez créer l’API "PulseManager", une application backend complète pour gérer des utilisateurs et projets. Ce projet fil rouge intégrera :

  • Node.js et Express
  • MySQL avec `mysql2/promise` et Sequelize ORM
  • MongoDB avec Mongoose
  • Architecture MVC : routes, controllers, models
  • CRUD complet, validation et gestion des erreurs
  • Tests Postman

Objectifs pédagogiques

  1. Installer et configurer l’environnement
  2. Comprendre la structure MVC
  3. Appliquer CRUD sur MySQL et MongoDB
  4. Valider et sécuriser les données
  5. Livrer un projet complet fonctionnel

Livrable final

Une API REST complète, multi-base (MySQL, Sequelize, MongoDB), testée avec Postman, prête pour production.
