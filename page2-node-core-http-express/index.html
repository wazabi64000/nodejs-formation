Présentation du thème

Dans ce module, nous explorons le cœur de Node.js, la manipulation des fichiers avec le module FS, et la création d’un serveur HTTP capable de traiter les requêtes GET et POST.

C’est quoi Node Core ?

Node Core regroupe les modules natifs fournis avec Node.js. Vous pouvez manipuler des fichiers, gérer le réseau, créer des serveurs HTTP, travailler avec les flux et événements, sans installer de bibliothèques externes.

Pourquoi apprendre HTTP avec Node ?

HTTP est le protocole de communication standard du web. Savoir créer un serveur HTTP avec Node permet de comprendre le fonctionnement interne des API et de préparer l’apprentissage d’Express.js et des frameworks modernes.

Comment ça fonctionne ?

Vous allez apprendre à utiliser les modules `fs`, `http` et `path`. Nous verrons comment créer un serveur simple, lire des fichiers statiques et traiter les requêtes GET et POST.

Objectifs pédagogiques

  • Maîtriser les modules Node Core (FS, Path, HTTP)
  • Créer un serveur HTTP basique
  • Gérer les requêtes GET et POST
  • Comprendre les concepts de flux et événements
  • Préparer l’apprentissage d’Express.js
