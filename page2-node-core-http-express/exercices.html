Exercice 1 : Serveur HTTP basique

Créez un serveur HTTP qui renvoie "Hello Node Core!" sur le port 3000.

Exercice 2 : Lire un fichier texte

Créez un fichier data.txt et affichez son contenu dans la console à l’aide de FS.

Exercice 3 : Chemins dynamiques

Utilisez Path pour construire un chemin absolu vers data.txt et affichez-le.

Exercice 4 : Paramètres GET

Créez un serveur HTTP qui récupère un paramètre "name" depuis l’URL et l’affiche.

Exercice 5 : Requête POST simple

Créez un serveur HTTP qui reçoit des données POST et les renvoie dans la réponse.
