Module FS – File System

Le module `fs` permet de lire, écrire, modifier et supprimer des fichiers. Il propose des méthodes synchrones et asynchrones pour gérer le système de fichiers.

Vous apprendrez à créer, lire et supprimer des fichiers, et à organiser vos données côté serveur.

Module Path

Le module `path` permet de gérer les chemins de fichiers et dossiers de manière fiable, quelle que soit la plateforme.

Vous verrez comment construire des chemins absolus et relatifs pour éviter les erreurs dans vos applications.

Module HTTP

Le module `http` permet de créer des serveurs web sans framework. Nous apprendrons à écouter un port, répondre à des requêtes et gérer différents types de réponses.

Requêtes GET et POST

GET permet de récupérer des données, POST de les envoyer. Vous apprendrez à analyser les données envoyées par le client, à traiter les formulaires et à renvoyer des réponses appropriées.

Flux et événements

Node.js utilise un modèle événementiel et des flux (streams) pour traiter les données efficacement. Vous verrez comment lire et écrire de gros fichiers sans bloquer le serveur.

Bonnes pratiques

Organisez votre code par modules, utilisez des fonctions réutilisables et commentez chaque section. Cela facilitera la transition vers Express.js et la création d’API robustes.
