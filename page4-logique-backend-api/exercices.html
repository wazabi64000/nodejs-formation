Module 4 – Exercices

Exercice 1 : MySQL direct

Créer une table `users` et écrire une route GET pour renvoyer tous les utilisateurs.

Exercice 2 : Sequelize ORM

Créer un modèle `User` avec Sequelize et écrire une route POST pour ajouter un utilisateur.

Exercice 3 : MongoDB avec Mongoose

Créer un modèle `User` Mongoose et implémenter une route GET pour récupérer tous les utilisateurs.

Exercice 4 : Validation et erreurs

Ajouter la validation sur tous les champs et gérer les erreurs avec les bons codes HTTP.

Exercice 5 : Test complet

Tester toutes les routes GET, POST, PUT et DELETE avec Postman pour MySQL, Sequelize et MongoDB.
