Présentation du module

Dans ce module, vous allez apprendre à créer des APIs REST complètes en utilisant différentes bases de données : MySQL (direct + Sequelize) et MongoDB (Mongoose).

Objectifs pédagogiques

  • Manipuler MySQL avec `mysql2/promise`
  • Utiliser Sequelize pour ORM relationnel
  • Créer des modèles MongoDB avec Mongoose
  • Développer des endpoints RESTful GET/POST/PUT/DELETE
  • Valider les données et gérer les erreurs correctement
