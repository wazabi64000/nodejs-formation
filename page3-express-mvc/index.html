Présentation du thème

Dans ce module, nous découvrons Express.js et l’architecture MVC (Model-View-Controller) pour structurer les applications Node.js professionnelles.

C’est quoi Express.js ?

Express est un framework minimaliste pour Node.js qui facilite la création de serveurs, la gestion des routes, les middlewares et les API REST.

Pourquoi MVC avec Express ?

L’architecture MVC sépare les responsabilités : les routes contrôlent la navigation, les controllers contiennent la logique métier, et les models gèrent les données. Cela rend le code plus lisible, testable et maintenable.

Objectifs pédagogiques

  • Installer et configurer Express.js
  • Créer des routes GET et POST
  • Utiliser controllers et models
  • Configurer la connexion à une base de données via `.env`
  • Tester vos endpoints avec Postman
