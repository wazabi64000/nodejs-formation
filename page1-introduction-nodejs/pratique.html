1. Installation de Node.js

  1. Téléchargez Node.js depuis nodejs.org et installez-le.
  2. Vérifiez l'installation : node -v et npm -v.
  3. Créez un dossier de projet et initialisez-le : npm init -y.

2. Premier script avec CommonJS (Require)

// math.js
function add(a, b) {
  return a + b;
}
module.exports = { add };

// app.js
const { add } = require('./math.js');
console.log(add(5, 7)); // Affiche 12

3. Premier script avec ESM (Import/Export)

// math.mjs
export function add(a, b) {
  return a + b;
}

// app.mjs
import { add } from './math.mjs';
console.log(add(5, 7)); // Affiche 12

Pour utiliser ESM, ajoutez `"type": "module"` dans votre package.json ou utilisez l’extension .mjs.

4. Utiliser Nodemon

Nodemon permet de relancer automatiquement vos scripts lors de changements :

npm install -g nodemon
nodemon app.js

5. Variables d’environnement avec .env

Installez dotenv :

npm install dotenv

Créez un fichier .env :

PORT=3000
DB_USER=root
DB_PASS=secret

Utilisez-le dans votre code :

import 'dotenv/config';
console.log(process.env.PORT); // 3000
