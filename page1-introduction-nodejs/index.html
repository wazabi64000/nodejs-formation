Présentation du thème

Dans ce premier module, vous allez découvrir Node.js, comprendre son fonctionnement et créer vos premiers scripts côté serveur.

C’est quoi Node.js ?

Node.js est un environnement d’exécution JavaScript côté serveur basé sur le moteur V8 de Google Chrome. Il permet d’exécuter du JavaScript hors du navigateur et d’interagir avec le système, les fichiers et le réseau.

Pourquoi utiliser Node.js ?

Node.js est conçu pour créer des applications performantes et scalables. Son architecture événementielle et non bloquante est idéale pour les API, les applications temps réel et les services web rapides.

Comment ça fonctionne ?

Node.js utilise le moteur V8 pour exécuter le JavaScript côté serveur. Il fournit des modules natifs pour gérer les fichiers, le réseau, HTTP et les processus. Vous pouvez également installer des modules tiers via NPM.

Installation

  1. Téléchargez Node.js depuis nodejs.org.
  2. Vérifiez l'installation : node -v et npm -v.
  3. Créez un dossier de projet et initialisez-le : npm init -y.

Exemple avec require (CommonJS)

// fichier math.js
function add(a, b) { return a + b; }
module.exports = { add };

// fichier app.js
const { add } = require('./math.js');
console.log(add(2, 3)); // 5

Exemple avec import/export (ESM)

// fichier math.mjs
export function add(a, b) { return a + b; }

// fichier app.mjs
import { add } from './math.mjs';
console.log(add(2, 3)); // 5

ESM permet d’utiliser la syntaxe moderne import/export et facilite la modularité dans les projets Node.js récents.

Objectifs pédagogiques

  • Comprendre ce qu’est Node.js et son fonctionnement
  • Installer Node.js et préparer l’environnement
  • Distinguer Require (CommonJS) et ESM
  • Exécuter des scripts simples côté serveur
  • Se familiariser avec les bonnes pratiques de modularité
