Exercice 1 : Premier script Node.js

Créez un fichier hello.js et affichez "Bonjour Node.js !" dans la console.

Exercice 2 : Module CommonJS

Créez un fichier math.js avec une fonction multiply(a, b) et importez-la dans app.js pour afficher le résultat.

Exercice 3 : Module ESM

Convertissez l’exercice précédent pour utiliser ESM (export/import).

Exercice 4 : Variables d’environnement

Créez un fichier .env contenant APP_NAME=NodeBootcamp et affichez cette variable dans index.js.

Exercice 5 : Script combiné

Créez un module qui exporte deux fonctions (add et multiply) et utilisez-les dans un script principal pour afficher les résultats.
